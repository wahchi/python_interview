# python 面试题

包安装
```
pip3 install -r requirement.txt
```

运行我写的函数
```
python3 answer.py
```

运行单元测试
```
pytest test_question.py
```

注意：

 - 你需要按照指定的接口，指定返回
 - 注意performance
 - 留心不同的test case
 - 需要python3环境
